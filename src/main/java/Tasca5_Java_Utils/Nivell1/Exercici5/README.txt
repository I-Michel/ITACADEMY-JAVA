*******************************************************************
INSTRUCCIONES PARA EJECUTAR EL PROGRAMA DESDE LA LÍNEA DE COMANDOS:
*******************************************************************

1. Abrir el Símbolo del sistema o Command Prompt (cmd) de Windows.
   (Se puede encontrar fácilmente tecleando "cmd" en el buscador de Windows.)

2. Una vez tengamos la ventana del cmd abierta, debemos introducir la
   palabra "cd" seguida de un espacio en blanco y la ruta raiz donde se
   encuentre el programa que queremos ejecutar y pulsar intro.
   Por ejemplo:
   cd C:\Users\ingri\IdeaProjects\Sprint1\src\main\java

3. Luego compilamos las clases escribiendo el siguiente comando y pulsando intro:
   javac Tasca5_Java_Utils\Nivell1\Exercici5\Main.java Tasca5_Java_Utils\Nivell1\Exercici5\Registro.java

4. Finalmente, ejecutamos el programa con el siguiente comando:
   java -cp . Tasca5_Java_Utils.Nivell1.Exercici5.Main

5. Pulsamos intro y el programa se ejecutará. En este caso, debemos
   seguir las instrucciones del programa.

   ¡Gracias por llegar hasta aquí!
