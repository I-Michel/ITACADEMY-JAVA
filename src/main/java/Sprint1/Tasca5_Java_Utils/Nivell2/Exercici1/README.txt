*******************************************************************
INSTRUCCIONES PARA EJECUTAR EL PROGRAMA DESDE LA LÍNEA DE COMANDOS:
*******************************************************************

1. Abrir el Símbolo del sistema o Command Prompt (cmd) de Windows.
   (Se puede encontrar fácilmente tecleando "cmd" en el buscador de Windows.)

2. Una vez tengamos la ventana del cmd abierta, debemos introducir la
   palabra "java" seguida de un espacio en blanco y la ruta donde se
   encuentre el programa que queremos ejecutar. Por ejemplo:
   java C:\Users\ingri\IdeaProjects\ITAcademy_JavaSpringBoot\src\main\java\Sprint1\Tasca5_Java_Utils\Nivell2\Exercici1\Main.java

3. Pulsamos intro y el programa se ejecutará. Nos creará un archivo.txt
   con el árbol de directorios ordenado alfabéticamente.

   ¡Gracias por llegar hasta aquí!
