En tots els exercicis, crea un projecte Gradle amb una classe Main que
demostri l'ús del patró (amb les invocacions que es necessitin).

OBSERVER:

Dissenya un sistema en el qual un Agent de Borsa (Observable) notifica a
diverses agències de Borsa (Observers) canvis quan la Borsa puja o baixa.

- És necessari que l'objecte Observable mantingui referències als Observers.
