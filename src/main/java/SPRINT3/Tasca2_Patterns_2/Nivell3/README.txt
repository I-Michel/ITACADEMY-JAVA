**********************************************************************
En tots els exercicis, crea un projecte Gradle amb una classe Main que
demostri l'ús del patró (amb les invocacions que es necessitin).
**********************************************************************

DEPENDENCY INJECTION:

Dissenya una classe que mostri en pantalla el preu de diversos articles.
Ja que haurà de mostrar-los també en diversos tipus de moneda.

Assegura't d'afegir-li com a Injecció de Dependència una classe Convertidor de
Moneda que efectuï la correcció del preu en funció del canvi de divisa.
