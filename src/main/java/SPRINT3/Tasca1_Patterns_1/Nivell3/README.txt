COMMAND:

Dissenya un pàrquing de 4 vehicles: un cotxe, una bicicleta, un avió i un vaixell.

Mostra com funciona el patró Command que implementa els mètodes arrencar, accelerar
i frenar per a cada tipus de vehicle.
